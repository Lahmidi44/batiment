Ouvrir & Exécuter dans Google Colab

Comment générer votre rapport en ligne

1
Ouvrir dans Colab

Cliquez sur le bouton jaune ci-dessus. Le notebook s'ouvre directement dans Google Colab – aucune installation requise.

2
Configurer les paramètres

Dans la cellule PARAMÈTRES, modifiez les coordonnées, le nom du site, la période et les seuils d'analyse selon votre besoin.

3
Exécuter & Publier

Cliquez sur Exécution → Tout exécuter. La dernière cellule génère ce site et le publie automatiquement ici via l'API GitHub.

Token GitHub requis pour la publication automatique :
  1. Allez sur github.com/settings/tokens et créez un token avec le scope repo.
  2. Dans Colab : Paramètres → Secrets, ajoutez le secret nommé GITHUB_TOKEN.
  3. Exécutez la cellule Déploiement GitHub Pages dans le notebook.
